# Interact Ville 2

## ストーリー
あなたは、森に住んでいるハムスターです。
友達のハムスターである「ともハム」が同じ森に住んでいます。
ともハムとは仲良しで、お互いにおしゃべりするのが大好きです。
住んでいる森では、りんごが通貨になります。
ともハムは家に家具をまだあまり置いていないので家具がほしいかもしれません。

## やってみよう
- ともハムの家には時計があるから、時間を覚えているよ
  - 君といつ話したかを覚えているかも...?
- ともハムは会話をした後に日記をつけてくれるよ
  - タイトル画面でログインすると一番新しい日記が見れるよ
- りんごを拾って、森のショップでアイテムを購入してみよう
- ともハムに家具をプレゼントすると、家においてくれるかも...?
- 親密度が上がると、ともハムと話せる回数が増えるかも...?



## 今後の機能追加や不具合修正予定

### 機能追加
- ログインはパスワードを必要とすること(2026/01/11)
- 食べ物をともハムに渡すとそれに応じてともハムの機嫌や会話内容が変化すること(2026/01/11)

### 不具合
- PCのWebブラウザで全画面表示の時にNow Loadingの文字が見切れること(2026/01/11)
  - 暫定対応として真ん中付近に移動(2026/01/12)
- PCから日本語入力ができないこと(2026/01/11)
- [Claude API](https://platform.claude.com/)をフロントから叩こうとしていて、セキュリティ面でブロックされること(2026/01/11)
  - ともハムと会話することができない
  - プロキシサーバを介するように変更
