# Interact Ville 2

## ストーリー
あなたは、森に住んでいるハムスターです。
友達のハムスターである「ともハム」が同じ森に住んでいます。
ともハムとは仲良しで、お互いにおしゃべりするのが大好きです。
住んでいる森では、りんごが通貨になります。

## はじめかた

1. はじめてのプレイの時は、設定画面からClaudeのAPIキーを入力してください。

※入力したAPIキーは、ブラウザ(クライアント側)に保存されるため、サーバ管理者には送信されません。

2. 好きなユーザ名を入力してログインボタンを押してください。

## 今後の機能追加や不具合修正予定

### 機能追加
- ログインはパスワードを必要とすること(2026/01/12)
- 食べ物をともハムに渡すとそれに応じてともハムの機嫌や会話内容が変化すること(2026/01/12)

- PCのWebブラウザで全画面表示の時にNow Loadingの文字が見切れること(2026/01/12)
- PCから日本語入力ができないこと(2026/01/12)
- Claude APIをフロントから叩こうとしていて、セキュリティ面でブロックされること(2026/01/12)
  - ともハムと会話することができない
